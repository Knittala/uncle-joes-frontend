# Uncle Joe's Frontend

A customer-facing web application for Uncle Joe's digital platform project.

## Overview
This frontend allows customers to:
- Browse the Uncle Joe's menu
- Log in to their account
- View their order history
- Check their Coffee Club points balance

## Tech Stack
- Vue.js

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository
   git clone https://github.com/Knittala/uncle-joes-frontend.git
2. Navigate into the project
   cd uncle-joes-frontend
3. Install dependencies
   npm install
4. Start the development server
   npm start

## Features
- **Menu Browsing** – Customers can explore the full Uncle Joe's menu
- **User Authentication** – Secure login for registered customers
- **Order History** – View past orders and details
- **Coffee Club** – Check loyalty points balance
